CSS - Tutorial

Cascading Stylee Sheet

På den här sidan kommer vi gå igenom lite grunder i CSS och vad man kan göra. Det är ett språk som ändrar utseende på en webbsida skriven i ett markup språk t.ex HTML. Nedan kan du lära dig hur man ändrar färg, byter placering på något, ändrar form på ett objekt etc.

Ändra bakgrundsfärg

Med CSS kan man ändra färger på backgrunden. Testa att trycka på knapparna nedan för att ändra backgrundsfärgen på rutan nedan.

div{ background-color: }

Animering

Med attributet @keyframes kan man animera i CSS.

.animation { width: 100px; height: 100px; border-radius: 50%; background: #0171bf; position: relative; -webkit-animation: mymove 5s infinite; animation: mymove 5s infinite; } @keyframes mymove { from { left: 0px; } to { left: 500px; } }
